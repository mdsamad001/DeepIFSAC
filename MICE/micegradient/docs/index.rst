Welcome to miceforest's documentation!
======================================

**miceforest** performs Multiple Imputation by Chained Equations (MICE) with random forests.
The main offerings of this package include:

- Fill in (impute) missing values in a dataset with MICE
- Diagnostic plots to explain imputation behavior and validity
- Impute new data using models obtained while running MICE.
